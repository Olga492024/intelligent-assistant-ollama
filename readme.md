Интеллектуальный ассистент-исследователь с RAG, Guardrails и многоагентной проверкой
Описание проекта
Данный проект представляет собой вопросно-ответную систему на основе локальной большой языковой модели (Ollama + Mistral), дополненную механизмом **RAG** (Retrieval-Augmented Generation) для поиска ответов в пользовательской базе знаний. База знаний состоит из 50 сгенерированных документов на русском языке, охватывающих различные аспекты искусственного интеллекта.

Для повышения надёжности, безопасности и этичности взаимодействия реализованы:
- Guardrails – фильтрация входящих запросов и исходящих ответов по спискам запрещённых тем и слов.
- Многоагентная архитектура – агент-исследователь (Researcher) генерирует ответ с пошаговым рассуждением (Chain-of-Thought), агент-фактчекер (FactChecker) проверяет ответ на соответствие извлечённому контексту.
- Логирование всех взаимодействий в структурированный JSONL-файл.
- Управление параметрами генерации – демонстрация влияния температуры (temperature) на стиль и креативность ответа.

Используемые технологии

- Python 3.11
- Ollama – локальный сервер для запуска LLM (модели: mistral:latest, nomic-embed-text)
- LangChain + LangChain-Ollama – построение агентов и цепочек
- ChromaDB – векторное хранилище для RAG
- Jupyter Notebook – основная среда демонстрации и анализа

Установка и запуск

1. Клонирование репозитория
bash
git clone <URL_репозитория>
cd intelligent-assistant-ollama
2. Создание виртуального окружения и установка зависимостей
python -m venv venv
source venv/bin/activate      # Linux / macOS
venv\Scripts\activate         # Windows
pip install -r requirements.txt
3. Установка и настройка Ollama
Скачайте и установите Ollama для вашей ОС.
Загрузите:
ollama pull mistral:latest
ollama pull nomic-embed-text
4. Выполните в корне проекта:
python generate_docs.py
5. Откройте demo.ipynb и выполните ячейки.

Полный анализ представлен в конце ноутбука demo.ipynb.
